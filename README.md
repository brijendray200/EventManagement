# Event Management System

A full-stack event platform for attendees, organizers, and admins. The project includes event discovery, role-based authentication, bookings, payment confirmation, notifications, organizer tools, and admin controls.

## Highlights
- React and TypeScript frontend powered by Vite
- Node.js and Express backend with MongoDB
- JWT authentication and Google OAuth support
- Organizer dashboard for creating events and reviewing attendees
- Booking and Razorpay payment workflow
- Notifications, profile management, and AI concierge support

## Project Structure
- `src/`: frontend application
- `server/src/`: backend API and business logic
- `public/`: static assets

## Run Locally

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm run dev
```
